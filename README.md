# Разработка

## Зачем?
Лаборатории аллергологии в медицинском центре "XXI век" требуется приложение для поддержки процесса анализа на аллергию. Старое приложение, написанное на Delphi 7 обладает рядом недостатков: ошибки при работе с базой данных, отсутствие документации, трудность поддержки, трудновоспроизводимые ошибки при работе, проблемы с печатью на современном принтере, абсолютная непереносимость на другие компьютеры.
